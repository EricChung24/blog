LINE 跟 IG 要收費了，然後呢？

社群平台接連推出付費版，免費的東西終究要還。資料掌握在別人手裡，連說話的地方都是租來的——這是我開部落格的原因。

Apr 21, 2026
閱讀時間: 4 分鐘

最近看到消息，LINE 跟 Instagram 陸續推出付費訂閱方案。

有人在討論值不值得訂，有人說「免費就夠用了」，有人說「早就預料到了」。

我的第一個反應不是要不要訂，而是：

這件事讓我有點不舒服，但我說不清楚哪裡不對。

然後我想了很久，才理清楚那個不舒服是什麼。

免費，從來都不是真的免費

LINE、Instagram、Facebook、Twitter——這些平台都是「免費」的。

但這個免費背後有一個我們隱隱知道、卻不太正視的交換：

你不是用戶，你是產品。

你的每一個按讚、每一秒停留、每一次搜尋，都在餵養廣告機器。

它知道你幾歲、你住哪裡、你最近在想什麼、你可能買什麼、你在乎的人是誰。

這些資料不是你的。是它的。

付費版出現，才讓人看清楚遊戲規則

付費版上線，反而讓事情更清晰：

它在問你：「你願意用錢換資料隱私嗎？」

有點諷刺——你的資料本來就應該是你的，現在它要你付費才能拿回「更少被追蹤」的使用體驗。

這不是升級服務，是把原本應該屬於你的東西包裝成產品賣回給你。

Twitter（現在叫 X）、Meta、LINE，一個一個往這條路走。

不是說他們做錯了——商業邏輯很清楚，維運有成本。

但從使用者的角度，這一刻很難不有點悵然：

我們以為免費得到了一個地方說話、留記錄、跟朋友連絡——但從來沒有真正「擁有」過它。

突然發現，我們的所有東西都是租來的

想想看：

  • 你在 Instagram 發了幾千張照片，但那些照片其實在 Meta 的伺服器
  • 你在 LINE 跟家人朋友的對話，在 LINE 的資料庫裡
  • 你在 Facebook 寫的文章，格式是 Facebook 定的，能不能轉出去是 Facebook 說了算

今天它改演算法，你的貼文沒人看。

今天它要你付費，你不付就功能縮水。

今天它倒了（或者政策改了），你什麼都帶不走。

我不是要製造恐慌，只是這件事第一次這麼具體地讓我覺得：

在別人蓋的地方說話，始終是租客。

這是我開部落格的原因

說到這裡，才輪到我自己的故事。

我開這個部落格，一開始的理由是「想寫東西」。

但現在我知道更深的理由是：

我不想把自己說過的每一句話、學到的每一件事，都交給某個平台保管。

部落格的文章，放在我自己控制的 GitHub Pages。

格式是 Markdown，一個純文字格式，不依賴任何公司。

哪天 GitHub 關掉，我有所有原始檔；哪天 Hugo 不流行了，我的文字依然是我的文字。

沒有演算法決定你的文章值不值得被看見。

沒有平台決定你能不能用全部的功能。

沒有人把你的讀者資料拿去賣廣告。

就只是一個地方，讓我可以寫，讓你可以讀，如果你覺得有用那很好，如果你不覺得也沒關係。

商業化本身不是問題

我要說清楚：我不反對商業化。

平台需要錢來維持運作，開發者需要收入，這是很正常的事。

訂閱制比廣告制，在某種程度上甚至更誠實——我付錢，你給我服務，清清楚楚。

我不舒服的，不是「要收費」，是那個轉變的方式：

先養成你的習慣、讓你依賴、讓你的記憶和人際關係都累積在這裡——

然後，在你已經無法輕易離開的時候，才慢慢改變條件。

這在商業上叫做 lock-in，很有效，但對使用者不太公平。

不想盲從，但也不是勒德分子

我沒有刪掉 LINE，也沒有刪掉 IG。

因為它們仍然是聯絡朋友、接收訊息的地方，完全放棄太不現實。

但我開始思考一件事：

哪些事情值得放在「自己擁有」的地方？

  • 我學到的東西 → 部落格、Obsidian（本地）
  • 我的工作記錄 → 自己的資料夾、Git
  • 重要的對話紀錄 → 定期備份
  • 長期想留存的文字 → 部落格

而社群媒體，就讓它繼續做它擅長的事：即時溝通、娛樂、快速分享。

只是不再把它當成「我的資料儲存庫」。

如果有一件事你可以從這篇帶走

那就是這個問題：

你現在最重要的資料、文字、記憶，放在哪裡？那個地方是你真正擁有的嗎？

不一定要開部落格，不一定要學 Git。

但值得想一想：如果那個平台明天消失、或者明天改變規則——你能帶走什麼？

我想繼續用 LINE 跟朋友說垃圾話。

我也想繼續在這裡，寫一些我真正想留下來的東西。

兩件事不衝突。

只是從今以後，我會更清楚，哪些是租來的地方，哪些是自己的地方。
